# IA-SOLUTION Vector

Landing page statique pour présenter `HCS-U7`, une couche d’authentification orientée biométrie cognitive, cryptographie post-quantique et entropy engine.

## Contenu du projet

- `index.html` : page unique statique

## Lancer en local

Comme le projet est entièrement statique, vous pouvez simplement ouvrir `index.html` dans un navigateur.

Si vous préférez utiliser un serveur local, vous pouvez par exemple lancer un serveur statique depuis ce dossier avec l’outil de votre choix.

## Déploiement

Le projet est compatible avec un déploiement direct sur Vercel.

### Vercel

Configuration recommandée :

- Framework Preset : `Other`
- Build Command : laisser vide
- Output Directory : laisser vide
- Root Directory : `/`
